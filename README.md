# Exploratory Data Analysis (EDA) - Class & Lab Tasks

This repository contains all lab assignments, classwork, and practical implementations for the Exploratory Data Analysis (EDA) course, conducted by **Prof. Ramesh C**.

## 📌 Overview
The contents of this repository demonstrate data preprocessing, statistical analysis, and visual data exploration techniques using Python. It serves as a comprehensive record of academic progress and hands-on laboratory tasks throughout the semester.

## 🛠️ Tech Stack & Libraries
* **Language:** Python 3
* **Data Manipulation:** NumPy, Pandas
* **Data Visualization:** Matplotlib, Seaborn, Plotly
* **Environment:** Jupyter Notebook / Google Colab

## 🚀 Getting Started

### Prerequisites
Ensure you have Python installed. You can install the required dependencies using pip:
```bash
pip install -r requirements.txt
```
